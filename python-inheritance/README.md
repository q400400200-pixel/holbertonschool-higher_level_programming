 # Python - Inheritance

This project covers inheritance concepts in Python including superclasses, subclasses, method overriding, and built-in functions like `isinstance`, `issubclass`, `type`, and `super`.

## Requirements

* Python 3.8.5
* Ubuntu 20.04 LTS
* pycodestyle 2.7.*
* All files must be executable
* First line: `#!/usr/bin/python3`

## Tasks
